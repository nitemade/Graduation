核心系统详细设计
① 随机房间系统（作品集加分点）
不完全算法生成。
预制房间拼接（模块化生成）
每个房间定义：
出口方向
敌人生成点
特殊房间类型
战斗房
武器房
强化房
商店房
Boss房
生成流程：
Start Room
→ 随机普通房若干
→ 强化房
→ 武器房
→ Boss房

你可以做：
随机种子（Seed）
每次开局不同布局
这在作品集非常加分。

② 武器系统（核心亮点之一）
前期近战
后期职业改变
辅助武器自动射击
架构：
BaseWeapon
 ├── MeleeWeapon
 ├── RangedWeapon
 └── SupportWeapon

SupportWeapon 特点：
自动寻找最近敌人
独立攻击间隔
不占主武器CD
这样会非常专业。

③ 随机漂浮武器
设计为：
武器实体生成
随机品质（白绿蓝紫橙）
随机词条
例子：
+10% 攻击速度
+20% 暴击率
命中附带燃烧
可以做权重系统。
典型 Roguelike 构建。

④ 强化系统（核心玩法）
强化可以设计为：
伤害强化
攻速强化
范围强化
生命强化
技能强化

强化来源：
通关房间
强化房三选一
击杀精英怪

推荐做“三选一强化界面”，非常有 Roguelike 味道。
⑤ 职业分支系统
你说后期根据职业改变。
建议设计：
前期：统一近战
打到第3房间后：
选择职业
├── 战士（强化近战）
├── 射手（远程为主）
├── 工程师（辅助武器强化）
职业会：
改变主武器类型
改变成长曲线
改变技能
这会让游戏结构清晰。

⑥ 敌人系统
10–20个敌人同场。
必须做：
对象池
简单状态机
Idle
Chase
Attack
Dead

敌人类型建议：
普通近战
远程怪
冲锋怪
精英怪

⑦ Boss设计（作品集重点）
Boss 不要只是血厚。

推荐做：
多阶段Boss
Phase1：普通攻击
血量70% → Phase2
Phase2：范围技能
血量30% → Phase3
Phase3：狂暴

Boss机制越清晰，作品集越强。
⑧ 成长系统
局内成长：
伤害
生命
攻速
技能等级
局外成长（可选）：

天赋树
解锁职业

🚀 四、性能优化（作品集必做）

⭐ 对象池
子弹
敌人
